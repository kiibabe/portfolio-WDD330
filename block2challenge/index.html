Challenge Two

Project Overview:

Purpose

The project was to make a web app that allows users to log in to their Spotify account and get stats for their top/favorite artists and tracks using the Spotify Web APIs. Also, using html5 canvas, the user can save their stats to their computer by saving it as an image.

Purpose

The audience is any Spotify premium subscriber who wants to know what is their top artists/tracks for the past several years for fun. They can also share the result with their friends and families by downloading the image.

Data Sources

Spotify APIs (External API)

Module List

  • SpotifyService.js - calls Spotify API to get user stats
  • SpotifyController.js - uses SpotifyService to get data, and renders it to the screen using HTML
  • StatsController.js - uses html2 canvas to output stats as image and allow users to download it
  • LoginController.js - uses SpotifyService to get user tokens, and assists in login process
  • main.js - module serves as a main controller
  • utilities.js - uses to send the request
  • spotifyIndex.html - contains references to JS scripts, css stylesheet, and html elements so JS can render
  • styles.css - contains styles for index.html
