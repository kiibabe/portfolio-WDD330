Chapter 8: Transforms and Transitions

  • With CSS3, we can move things around without a line of JavaScript
  • Transforms:
    • This property you lets us translate, rotate, scale, and/or skew any element
    • Transform does not work on inline element
    • We can change the origin with transform-origin
    • Translation:
      • This function allows us to move elements left, right, up, or down
      • Moving an element 50px from the left and 50px to the top 
          transform: translate(50px, -50px);
    • Scale:
      • This function allows us to enlarge or shrink an element
      • scale(1): default, scale(2): doubled, scale(0.5): half
      • Enlarging an element by 1.5 * x and 0.25 * y 
          transform: scale(1.5, 0.25);
    • Rotate:
      • This function rotates an element around the point of origin by a specified angle value
      • Rotating an element by 90 degrees 
          transform: rotate(90deg);
    • Skew:
      • This function specifies a skew along the x and y axes 
          transform: skew(10deg, 10deg);
  • Transitions:
    • This property allows the values of CSS to change over time, providing simple animations
    • transition-duration
      • How long the transition will take
      • Set in s (seconds) or ms (milliseconds)
      •   transition-duration: 0.2s;
      • "200ms is generally considered the optimum time for a transition: anything slower will make the website seem slow, drawing generally unwanted attention to what was supposed to be a subtle effect. Anything faster may be too subtle."
    • transition-timing-function
      • This lets us control the pace of the transition more in detail
        • ease
        • ease-in
        • ease-out
        • linear
        • cubic-bezier
        • steps
    • transition-delay
    • Introduces a delay before the transition begins
    •   transition-delay: 50ms;
    • Keyframes:
    • To animate an element, first create a named animation, then attach it to an element
    •   @keyframes moveRight { 
        /* put animation keyframes here */
        from {
            transform: translateX(-50%);
        }
        to {
            transform: translateX(50%);
        }
    }
    @keyframes appearDisappear { 
        0%, 100% {
            opacity: 0;
        }
        20%, 80% {
            opacity: 1;
        }
    }
  • Animation properties
    • animation-name
    • animation-duration
    • animation-timing-function
    • animation-iteration-count
    • animation-direction
    • animation-delay
    • animation-fill-mode
    • animation-play-state

Chapter 12: Canvas, SVG, and Drag and Drop

  • Canvas
    • With HTML5’s Canvas API, we can draw anything we can imagine through JavaScript
    • * The text in between the canvas tags will only be visible if canvas is not supported by the viewer's browser 
        <canvas id="myCanvas" class="myCanvas" width="200" height="200">
        Sorry! Your browser doesn’t support Canvas.
    </canvas>
  • SVG:
    • Stands for Scalable Vector Graphics
    • Drawing a circle: 
        <svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 400 400">
                    <circle cx="50" cy="50" r="25" fill="black"/>
                </svg>
    • <desc> tag: Adds description
    • <rect> tag: Describe a rectangle
    • We can utilize tools like Inkscape and Raphaël to make it easier
  • Drag and Drop
    • This API allows you to specify that certain elements are draggable, and then specify what should happen when these draggable elements are dragged over or dropped onto other elements on the page
    • Adding draggable="true" attribute to an element
    • Draggable is not a Boolean attribute, so you have to set it exactly to true
