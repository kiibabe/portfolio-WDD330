Chapter 5

  • Object in JavaScript is a self contained set of related values and functions
  • Objects act as "a collection of named properties that map to any JavaScript value such as strings, numbers, booleans, arrays and functions.
  • A method is when a property’s value is a function
  • Objects are used to keep any related info and functionality together
  • Object literals example: 
         const ryan = {
                        name: 'Ryan',
                        'full name': 'Ryan Gosling',
                        height: 6,
                        weight: 168,
                        actor: true,
                        actress: false,
                        coster: ['Emma','Rachel','Harrison'],
                        dance() {
                            return 'Dance dance, la la land!';
                        }
                    };
  • The ability to create objects with computed property keys was introduced in ES6. This means that JavaScript code can be placed inside square brackets and the property key will be the return value of that code. 
         const emma = { name: 'Emma', ['catch' + 'Phrase']: 'Emma Stone' };
         << { name: 'Emma', catchPhrase: 'Emma Stone' }
  • To call an object’s method we can also use dot or bracket notation 
         ryan.dance()
    << 'Dance dance, la la land!'

    ryan['dance']()
    << 'Dance dance, la la land!'
  • Groups of characters can be placed together inside square brackets. 
         const vowels = /[aeiou]/

    /[A-Z]/    // All the uppercase letters

    /[0-9]/    // The digits 0-9

Chapter 6

  • "The Document Object Model (DOM) allows you to access elements of a web page and enable interaction with the page by adding and removing elements, changing the order, content and attributes of elements, and even altering how they are styled"
  • The Document Object Model (DOM) represents an HTML document as a network of connected nodes that form a tree-like structure. It treats everything on a web page as a node
  • The DOM stores any whitespace that is in the HTML document as text nodes, so there appear to be some extra text nodes
  • Console Shortcuts:
    • Pressing TAB will autocomplete any methods and should show you a list of possible methods
    • Pressing the UP arrow key will select the previous command entered.
  • Node Type Code
    • 1 element
    • 2 attribute
    • 3 text
    • 8 comment
    • 9 body
  • The document.getElementById(), document.getElementsByClassName(), document.getElementsByTagNames() and document.querySelector() can be used to access elements on a page.
  • The createElement() and createTextNode() methods can be used to create dynamic markup on the fly.

Chapter 7

  • Events are what provides the link between the web page and user interactions. Whenever the user interacts with a web page, an event occurs that our program can detect
  • Event Listener: 
         document.body.addEventListener("click", doSomething);
  • Inline Event Handler: 
         <p onclick="console.log('Clicked!')">Click Me</p>
  • Old Event Handler: 
         document.onclick = function (){ console.log('Clicked!'); }
  • Mouse Events:
    • click
    • mouseup
    • mousedown
    • dblclick (doubleclick)
    • mouseover
    • mouseout
    • mousemove
  • We cannot attach both a click and doubleclickevent to the same element because it’s impossible to tell if a click is the first click of a doubleclick or just a single click. This means that a doubleclickevent will alwayscause theclickevent to fire.
  • Keyboard Events:
    • touchstart
    • touchend
    • touchmove
    • touchenter
    • touchleave
    • touchcancel
  • Touch Events:
    • keydown
    • keyup
    • keypress
  • Removing an event Listener (removeEventListener() method): 
         const aParagraph = document.getElementById('a');
                            aParagraph.addEventListener('click', remove);
                            
                            function remove(event) {
                                console.log('removeEventListener() method example');
                                aParagraph.style.backgroundColor = 'blue';
                                aParagraph.removeEventListener('click',remove);
                            }
  • preventDefault()is a method of the event object that can be used inside the callback function to stop the default behavior. However, think twice before using the method. Users will expect certain behaviors, and preventing them may cause confusion.
